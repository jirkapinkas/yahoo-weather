<h1>Yahoo weather API app</h1>

<p>This project is to showcase Spring Boot, Spring Data JPA, REST web. services (XML and JSON) and Spring Cache.</p>

<p>
	How to run:
	<code>mvn jetty:run</code>
</p>

<p>
	How to build WAR file:
	<code>mvn package</code>
</p>

<p>
	You can deploy this web application to <a href="http://www.heroku.com"
		target="_blank">Heroku</a>. Live preview is here: <a
		href="http://test-yweather.herokuapp.com" target="_blank">http://test-yweather.herokuapp.com</a>
</p>
